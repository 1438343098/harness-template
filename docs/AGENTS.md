# docs/ Directory Navigation — AGENTS.md

> This directory holds all project documentation, including requirements documents and design documents.

---

## Directory Structure

```
docs/
├── prd/           # Product requirements documents (placed by user)
│   ├── AGENTS.md
│   ├── REQUIREMENTS_TEMPLATE.md  # Requirements document template
│   └── (user-placed requirements documents)
└── design/        # Design documents
    ├── AGENTS.md
    ├── DESIGN_INTAKE.md           # Design interpretation spec
    ├── assets/                    # User-placed design images
    └── extracted/                 # Design specs auto-generated by Claude
```

---

## When to Read This Directory

| Task | Files to read |
|------|----------|
| Parse requirements | User files under `docs/prd/` |
| Interpret design | Images in `docs/design/assets/` + `DESIGN_INTAKE.md` |
| View design spec | `docs/design/extracted/design-spec.md` |

---

## User Guide

- **Place requirements documents:** Put PRDs into `docs/prd/`, any name or format is fine
- **Place design images:** Put Figma-exported images into `docs/design/assets/`
- **View parsed specs:** See `docs/design/extracted/`
